제2장. HTML5: 웹 문서 만들기 / 2.1 텍스트 표현 / 2.1.1 줄을 바꾸는 br 요소

예제 2-1 줄바꿈을 수행하는 br 요소

<!DOCTYPE html>
<html>
    <head>
        <title> br 요소 </title>
    </head>
    <body>
        br 요소는 문서의 어느 곳에서든 강제로 줄바꿈을 수행한다. <br>
        여러 줄을 바꾸는 경우 원하는 개수만큼을 연속적으로 사용한다.
        <br/><br/><br/>
        br을 세 번 사용했으므로 위 문장 사이에는 빈 두 줄이 나타난다.
    </body>
</html>

실행 결과

br 요소 br 요소는 문서의 어느 곳에서든 강제로 줄바꿈을 수행한다.
여러 줄을 바꾸는 경우 원하는 개수만큼을 연속적으로 사용한다.


br을 세 번 사용했으므로 위 문장 사이에는 빈 두 줄이 나타난다.
